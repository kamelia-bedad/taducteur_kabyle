# Traducteur Kabyle ↔ Français

# Descript
Ceci est un petit projet en Python qui traduit des mots du Kabyle vers le Français et du Français vers le Kabyle.  
Le Kabyle est ma langue maternelle et j'ai voulu créer un traducteur simple pour montrer quelques mots avec leurs lettres berbères (tifinagh).

# Fonctionnalités
Traduction Kabyle → Français
Traduction Français → Kabyle
 Affichage du mot en lettres berbères (Tifinagh)
 Interface graphique simple avec Tkinter
